🌾 AgroManager - Gestão e Tecnologia no Campo

O AgroManager é um sistema web desenvolvido para auxiliar produtores rurais na gestão e análise de atividades agrícolas. Ele reúne ferramentas úteis como calculadora de lucro, calendário de safra e notícias do setor agro em uma interface moderna, responsiva e intuitiva.

🚀 Demonstração

Projeto em página única (SPA simples) com interface interativa voltada ao agronegócio.

✨ Funcionalidades
📊 Calculadora de Lucro
Simula o lucro por hectare
Entrada de custo, produção e preço da saca
Exibe:
Receita bruta
Lucro líquido
Status de lucro ou prejuízo
📅 Calendário de Safra
Lista culturas agrícolas
Mostra fases de:
Plantio
Colheita
Processamento
Informações organizadas em cards dinâmicos
📰 Notícias do Agro
Cards com notícias simuladas do setor agrícola
Destaque visual com ícones e categorias
📩 Formulário de Contato
Campos de nome, email, assunto e mensagem
Estrutura pronta para integração futura com backend
🎨 Interface moderna
Design responsivo
Tema escuro com estilo agrícola tecnológico
Animações suaves e efeitos visuais

HTML5
CSS3 (Flexbox + Grid + Variáveis CSS)
JavaScript Vanilla (puro)
